## Run Locally

Clone the project

```bash
  git clone https://github.com/rolando-r/prueEntityFramework/
```

Go to the project directory

```bash
  cd prueEntityFramework
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
